 Virtual Box

<br />
 
 XShell

<br />


## 1. docker-compose 설치하기
docker-compose를 설치합니다.
> docker-compose 설치
> ```
> apt-get update
> apt-get install docker-compose-plugin
> ```
> <br />
>
> docker-compose 설치 확인
> ```
> docker compose version
> ```
